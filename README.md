# NOVA: Next-gen Optimized Virtual Assistant

**NOVA** is a voice-based virtual assistant written in Python that can perform a variety of tasks including answering questions, opening applications, searching the web, and more — all through natural voice commands.

---

## 🎯 Features

* 🎤 Voice command recognition
* 🌐 Web search and Wikipedia summaries
* ⏰ Date and time reporting
* 📂 Open applications (Notepad, browser, etc.)
* 💬 Personalized responses with natural language
* 🛠️ Easily extendable for more functionalities

---

## 📦 Technologies Used

* **Python 3.8+**
* **SpeechRecognition** - for voice-to-text
* **Pyttsx3** - for text-to-speech
* **Wikipedia API** - for fetching quick summaries
* **Datetime, OS** - for utility functions

---

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/Magnus0969/NOVA.git
cd NOVA
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run NOVA

```bash
python nova.py
```

Speak commands like:

* "What time is it?"
* "Open Google"
* "Search for Elon Musk"
